# TIC-TAC-TOE-AI

![screenshot](screenshot.png)

# What it does

- A simple tic tac toe game to play against the computer
- It finds best moves using a minimax algorithm, so good luck trying to beat it

## Built With

- HTML5.
- CSS3.
- Bootstrap.
- React.js.
- JavaScript.
- AWS S3.

## Live Version
